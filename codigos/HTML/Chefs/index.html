Regras do jogo

Como jogar

> Embaralhe as cartas, distribua 3 para cada jogador e coloque o restante no meio do tabuleiro, com os ingredientes voltados para baixo.

> O primeiro jogador deve girar a roleta para dar início ao jogo. Cada jogador deve girar a roleta na sua vez e descartar uma carta após fazer o que caiu na roleta.

> O objetivo do jogo é juntar todos os ingredientes necessários para montar um dos pratos especificados no manual do jogo.

> Ganha aquele que juntar todos os ingredientes primeiro.
